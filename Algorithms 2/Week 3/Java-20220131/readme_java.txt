=======================================================
To compile the Java one file version:
=======================================================
javac BinarySearchTree.java

=======================================================
To compile the Java multiple file version:
=======================================================
javac *.java

To run one file version once compiled:

java BinarySearchTree

To run multiple file version once compiled:

java BinarySearchTreeTest

=======================================================
For Eclipse users and the Java multiple file version:
=======================================================

- Create a new java project
- right clicking the src folder
- select the option New > Package
- name the package bst
- drag and drop the files into the bst package inside of the package explorer. 
- add the following line to the top of every file:   package bst;

After that, it should compile and run when they click the run button.
